# 质量网关

## 输入依赖

- **必须**：SwRS 软件需求文档
- **必须**：FC 功能清单（--fc）
- **必须**：边界定义文档（--boundary）

## 检查维度

| 维度 | 检查项 | 判断标准 |
|------|--------|----------|
| **完整性** | 每条 FC-XXX 是否有对应 SwRS 覆盖 | 覆盖率 = 100% |
| **量化性** | 所有条件和参数是否量化 | 无模糊词 |
| **一致性** | 模块间接口定义是否匹配 | A 输出 = B 输入 |
| **可测试性** | 是否可设计测试用例验证 | 有明确的输入-条件-预期 |
| **ID 唯一性** | SwSR-ID 是否无重复 | 无重复编号 |

## 输出格式

### 问题汇总

| 问题ID | 类型 | 涉及需求ID | 问题描述 | 严重程度 | 建议修改 |
|--------|------|------------|----------|----------|----------|

严重程度：Critical / Major / Minor

### 覆盖率报告

| FC-ID | 总数 | 已覆盖 | 未覆盖 |
|-------|------|--------|--------|

### 总体评估

- [ ] 通过
- [ ] 需修改（Critical: X, Major: Y）

## 执行示例

```bash
/quality-gate ./SwRS.md --fc=./FC清单.md --boundary=./边界定义.md
```
