# 变更跟踪

## 变更来源

1. **系统部 CTS 变更**：需求本身变化
2. **开发发现技术不可行**：实现层面问题
3. **测试发现无法验证**：测试层面问题

## 变更分类

- **A类（重大）**：影响多个模块，需重新评审
- **B类（中等）**：影响单个模块，局部更新
- **C类（轻微）**：文档修正，不影响实现

## 输出格式

### 变更影响分析

| 项目 | 内容 |
|------|------|
| 变更单号 | CHG-XXX |
| 变更来源 | |
| 变更类型 | A/B/C |
| 紧急程度 | 高/中/低 |

### 影响范围

| 影响类型 | FC-ID | SwRS-ID | 模块 |
|----------|-------|----------|------|
